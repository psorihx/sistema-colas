# Sistema de Colas para Banco

## ¿Qué hace el proyecto?

Este proyecto es una aplicación web desarrollada con **Java, Spring Boot y MySQL** que permite gestionar el sistema de colas de un banco.

El sistema permite:

- Iniciar sesión.
- Registrar clientes.
- Seleccionar el tipo de trámite.
- Generar un número de ticket automáticamente.
- Visualizar el turno generado.
- Confirmar la asistencia del cliente.
- Administrar y actualizar el estado de los turnos.

---

## ¿Con qué se ejecuta?

Para ejecutar el proyecto se necesita:

- Java JDK 17 o superior.
- IntelliJ IDEA o Visual Studio Code.
- MySQL Server.
- Maven (incluido en el proyecto).

---

## ¿Cómo ejecutar el proyecto?

### IntelliJ IDEA

1. Abrir IntelliJ IDEA.
2. Seleccionar **Open** y abrir la carpeta del proyecto.
3. Ejecutar la clase **SistemaColasApplication.java**.
4. Abrir el navegador y acceder a:

```
http://localhost:8080
```
